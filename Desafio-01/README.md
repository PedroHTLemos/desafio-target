# Desafio Target 01 - Cálculo de Comissões

Este programa lê os registros de vendas presentes no arquivo vendas.json e calcula a comissão de cada vendedor seguindo as regras:

-	Vendas abaixo de R$100,00 não gera comissão
-	Vendas abaixo de R$500,00 gera 1% de comissão
-	A partir de R$500,00 gera 5% de comissão

Os resultados são exibidos no terminal de forma agrupada por vendedor.

# Como executar o programa
- Acesse a pasta Desafio01
- execute o programa via dotnet run

# Caso o usuário queira alterar os registros de vendas
- Acesse a pasta Desafio01/Data
- altere o registro no arquivo "vendas.json"
