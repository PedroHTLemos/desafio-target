# Desafio Target 02 - Sistema de Estoque

Este programa lê os registros de produtos presentes no arquivo "estoque.json" e permite realizar movimentações de entrada e saída no estoque.
As alterações são registradas no próprio arquivo json.

# Funcionalidades:

• Listagem de todos os produtos cadastrados
• Movimentação de estoque
• Validação para impedir que a saída seja maior que o estoque disponível
• Atualização automática no arquivo estoque.json após qualquer movimentação

Os resultados são exibidos diretamente no terminal.

# Como executar o programa
- Acesse a pasta Desafio02
- execute o programa via dotnet run

# Caso o usuário queira alterar os produtos no estoque
- Acesse a pasta Desafio02/Data
- altere o registro no arquivo "estoque.json"
