# Desafio Target 03 - Cálculo de Juros

Este programa calcula os juros de uma dívida com base em uma data de vencimento, considerando juros de 2,5% ao dia de atraso.

O usuário deve informar:
- Valor da dívida
- Data de vencimento (dd/mm/aaaa)

O programa exibe:
- Dias de atraso
- Valor dos juros
- Valor total a pagar

# Como executar o programa
- Acesse a pasta Desafio03
- Execute o programa via dotnet run
